# Aura Estética - Proyecto Final Talento Tech 2025

## Descripción

**Aura Estética** es una aplicación web desarrollada como parte del Proyecto Final del programa Talento Tech 2025. El proyecto tiene como objetivo crear una plataforma moderna y estética para la gestión de servicios de belleza y bienestar, utilizando tecnologías actuales como Vite, React y JavaScript.

## Tecnologías utilizadas

- **Vite**: Herramienta de construcción y desarrollo rápido para aplicaciones modernas.
- **React**: Biblioteca de JavaScript para construir interfaces de usuario.
- **JavaScript**: Lenguaje de programación principal.
- **HTML/CSS**: Estructura y estilo de la aplicación.
